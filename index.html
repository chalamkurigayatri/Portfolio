Latest Projects

Mechanic Master

Mechanic Master is an online platform built using HTML, CSS, and JavaScript that connects users with nearby mechanics via GPS. Both users and mechanics can register, log in, and interact. Users can locate services, buy fuel or spare parts, and easily contact mechanics during vehicle breakdowns.

Skin analysis

Skin Analysis and Product Recommendation System is an AI-based tool developed using CNN and machine learning algorithms. Users upload a skin image, and the system detects issues like acne, wrinkles, or spots. Based on the analysis, it recommends suitable skincare products for personalized treatment and care.

Cloth Overproduction

Cloth Overproduction Prediction System is a machine learning project built using Python. It applies feature engineering and predictive models to analyze production data and forecast optimal cloth output. This helps industries avoid overproduction, reduce waste, and optimize resources through data-driven decision-making and visual insights.

Amazon Clone

Amazon Clone is a static web project replicating the interface of the original Amazon platform. Built using HTML and CSS, it showcases a structured layout of an e-commerce site, including product listings, navigation bars, and design elements. Though non-responsive, it demonstrates front-end development skills effectively.
